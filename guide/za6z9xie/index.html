PlantUML

约 1467 字大约 5 分钟

2025-03-01

概述

主题支持在 文章中 嵌入由 PlantUML 。

该功能由 vuepress-plugin-md-enhance 提供支持。

配置

主题默认不启用该功能。

在 .vuepress/config.ts 配置文件中，启用该功能：

.vuepress/config.ts
export default defineUserConfig({
  theme: plumeTheme({
    markdown: {
      plantuml: true, 
    },
  })
})

注

以下文档 Fork 自 vuepress-theme-hope, 遵循 MIT 许可证。

格式

你可以插入plantuml 支持的相同内容，例如:

@startuml
内容
@enduml

示例

序列图

@startuml
Alice -> Bob: 认证请求

alt 成功情况

    Bob -> Alice: 认证接受

else 某种失败情况

    Bob -> Alice: 认证失败
    group 我自己的标签
    Alice -> Log : 开始记录攻击日志
        loop 1000次
            Alice -> Bob: DNS 攻击
        end
    Alice -> Log : 结束记录攻击日志
    end

else 另一种失败

   Bob -> Alice: 请重复

end
@enduml

@startuml

if (Page.onSecurityCheck) then (true)
  :Page.onInit();
  if (isForward?) then (no)
    :Process controls;
    if (continue processing?) then (no)
      stop
    endif

    if (isPost?) then (yes)
      :Page.onPost();
    else (no)
      :Page.onGet();
    endif
    :Page.onRender();
  endif
else (false)
endif

if (do redirect?) then (yes)
  :redirect process;
else
  if (do forward?) then (yes)
    :Forward request;
  else (no)
    :Render page template;
  endif
endif

stop

@enduml
